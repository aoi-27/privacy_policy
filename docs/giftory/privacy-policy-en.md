# Privacy Policy

[日本語](./privacy-policy.html)

Last Updated: February 15, 2025

## Introduction

"Giftory" (hereinafter referred to as "this app") is an application for managing important anniversaries and events. This Privacy Policy explains how we handle personal information in this app.

## Information We Collect

### 1. Photo Library Access

This app requests access to your photo library for the following purposes:

- **Photo Library (Read & Write)**: To select photos to associate with anniversaries and to customize background photos

Selected photos are processed as follows:
- **When using local storage only**: Photos are stored only on your device and are never sent to external servers
- **When cloud sync is enabled**: Thumbnails of selected photos may be stored in Firebase Cloud Storage

### 2. Authentication Information (Sign in with Apple)

When using the cloud sync feature, we authenticate using Sign in with Apple. This may collect the following information:

- User identifier associated with your Apple ID
- Email address (if you choose to share it)

This information is used for account management and cloud data synchronization.

### 3. Cloud Data (Firebase Firestore)

When cloud sync is enabled, the following data is stored in Firebase Firestore:

- Anniversary information (name, date, cycle, memo)
- Celebration memo records
- Category information (name, color)
- Photo thumbnail images
- App settings (display format, theme, etc.)

**Important**: Cloud sync only works when you explicitly enable it. By default, all data is stored only on your device.

### 4. Analytics and Crash Reports (Firebase)

This app uses the following services to improve our service:

- **Firebase Analytics**: To analyze app usage
- **Firebase Crashlytics**: To detect crashes and errors and improve app stability

These services may collect the following information:
- Device information (model, OS version)
- App usage information
- Crash logs and error information

### 5. Advertising Information (AdMob)

In the free version (local storage only), this app uses Google AdMob to display advertisements. AdMob may collect the following information to optimize ad delivery:

- Advertising Identifier (IDFA)
- Device information
- App usage information
- Ad interaction information

**Important**: Ads are not displayed when you are signed in with Sign in with Apple.

For more details, please see [Google's Privacy Policy](https://policies.google.com/privacy).

### 6. Widget Feature

This app provides a widget feature for your home screen. The widget uses App Groups to share data with the main app, but all data is processed only on your device.

## How We Use Information

Collected information is used for the following purposes:

1. Providing app functionality (anniversary management, photo association, cloud sync)
2. User account management (when using Sign in with Apple)
3. Improving app quality and fixing bugs
4. Service improvement based on analytics
5. Delivering advertisements (free version only)

## Information Sharing

Information collected by this app may be shared with the following third-party services:

- **Apple Inc.**: For Sign in with Apple authentication
- **Google Firebase**: For cloud data sync, analytics, and crash reporting
- **Google AdMob**: For advertising delivery (free version only)

These services handle information according to their respective privacy policies.

## Your Rights

You have the following rights:

- You can change photo library access permissions in device settings
- You can enable or disable cloud sync in app settings
- You can delete cloud-stored data using the account deletion feature
- You can limit the use of the advertising identifier (IDFA) in device settings (free version only)
- You can delete local data and stop future information collection by uninstalling this app

## Data Retention

- **Local data**: Stored on your device and retained until you uninstall the app
- **Cloud data**: Stored in Firebase Firestore until you delete your account
- **Analytics data**: Stored according to Firebase data retention policy
- **Advertising data**: Stored according to AdMob policy

## Data Deletion

You can delete data using the following methods:

- **Local data**: Deleted when you uninstall the app
- **Cloud data**: All data stored in Firestore is deleted when you execute "Delete Account" from the app settings screen

## Children's Privacy

This app is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

## Security

We implement the following security measures to protect your information:

- Secure authentication via Firebase Authentication
- Data protection through Firestore security rules
- App legitimacy verification via Firebase App Check
- Encrypted communication (HTTPS/TLS)

However, please understand that no method of transmission over the Internet or electronic storage is 100% secure.
