# プライバシーポリシー

[English](./privacy-policy-en.html)

最終更新日: 2026年6月28日

## はじめに

「Kilo-Log」（以下、「本アプリ」といいます）は、体重・食事を記録してグラフで推移を確認できる体重管理アプリです。本プライバシーポリシーでは、本アプリにおける個人情報の取り扱いについて説明します。

## 収集する情報

### 1. 体重・食事データ

本アプリは、体重・食事の記録・管理のため、以下のデータをデバイス内にのみ保存します：

- 体重・体脂肪率の記録（日付・数値）
- 気分（great / good / normal / poor）・メモ
- 食事記録（食事名・カロリー・食事タイプ）
- 目標体重・開始体重・達成期限

これらのデータはお客様のデバイス内（SwiftData）にのみ保存され、外部サーバーへ送信されることは一切ありません。

### 2. HealthKit データ

本アプリは、体重データをAppleヘルスケアと連携するため、お客様の許可を得た上でHealthKitにアクセスします：

- **読み取り**: 過去の体重データの一括インポート
- **書き込み**: 記録した体重をヘルスケアアプリへ保存

取得したHealthKitデータを外部サーバーへ送信することは一切ありません。HealthKit連携は任意の機能であり、許可しない場合でもアプリ内での記録・管理はご利用いただけます。

### 3. 生体認証・PINロック

本アプリは、アプリロック機能のために以下を使用します：

- **Face ID / Touch ID**: LocalAuthentication Framework を使用した生体認証（デバイス内処理のみ）
- **PINコード**: 4桁のPINをKeychain（iOSの安全な認証情報ストレージ）に保存

これらの認証情報は外部サーバーへ送信されることは一切ありません。

### 4. 広告関連情報（AdMob）

本アプリは、Google AdMobを使用してバナー広告を表示しています。AdMobは、広告配信の最適化のために以下の情報を収集する場合があります：

- 広告識別子（IDFA）
- デバイス情報
- アプリの使用状況
- 広告のインタラクション情報

なお、本アプリは起動時にGoogle UMP（ユーザーメッセージングプラットフォーム）による同意確認を行い、同意後にAppleのApp Tracking Transparency（ATT）許可ダイアログを表示します。

詳細については、[Googleのプライバシーポリシー](https://policies.google.com/privacy)をご覧ください。

## 情報の利用目的

収集した情報は、以下の目的で利用されます：

1. アプリの機能提供（体重・食事の記録・管理・グラフ表示）
2. HealthKit連携による体重データの同期
3. アプリロック機能の提供
4. アプリの品質向上およびバグ修正
5. 広告の配信

## 情報の共有

本アプリが収集する情報は、以下の第三者サービスと共有される場合があります：

- **Google AdMob**: 広告配信のため

体重・食事データ、HealthKitデータ、生体認証情報およびPINは第三者と共有されることはありません。

## お客様の権利

お客様は、以下の権利を有します：

- デバイスの設定から、HealthKitデータへのアクセス許可（読み取り・書き込み）を変更できます
- デバイスの設定から、広告識別子（IDFA）の利用を制限できます
- アプリの設定から、AppLockを無効化またはPINを変更できます
- 本アプリをアンインストールすることで、今後の情報収集を停止できます（Keychain内のPINも削除されます）

## データの保存期間

- 体重・食事データ: デバイス内にのみ保存され、アプリが外部に送信することはありません
- HealthKitデータ: デバイス内でのみ参照・書き込みされ、外部送信は行いません
- PINコード: Keychainに保存され、アプリアンインストール時に削除されます
- 広告データ: AdMobのポリシーに従って保存されます

## 子供のプライバシー

本アプリは13歳未満の子供を対象としていません。13歳未満の子供から故意に個人情報を収集することはありません。

## セキュリティ

お客様の情報を保護するため、適切な技術的および組織的なセキュリティ対策を実施しています。PINコードはiOSのKeychainに安全に保存されます。ただし、インターネットを介した送信方法や電子的な保存方法は100%安全ではないことをご理解ください。

## プライバシーポリシーの変更

本プライバシーポリシーは、予告なく変更されることがあります。重要な変更がある場合は、本ページの「最終更新日」を更新します。定期的にご確認いただくことをお勧めします。
